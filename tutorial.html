    return False  # Wrong!

üõ°Ô∏è Why This is Safe

üìñ Glossary - Important Words

AJAX

Sending data to the server without refreshing the page. Like sending a text message instead of making a phone call.

API

Application Programming Interface - How programs talk to each other. Like a waiter carrying orders between you and the kitchen.

Authentication

Proving who you are. Like showing your ID or entering a password.

Backend

The server/brain of the application that users don't see. Where the real work happens.

Browser

The program you use to view websites (Chrome, Safari, Firefox).

CSS

Cascading Style Sheets - Makes websites look pretty with colors, fonts, and layouts.

Database

Where we store information permanently. Like a huge filing cabinet.

Endpoint

A URL where the API listens for requests. Like different doors in a building.

Frontend

What users see and interact with. The visual part of the website.

Function

A reusable piece of code. Like a recipe you can follow multiple times.

HTML

HyperText Markup Language - The structure/skeleton of web pages. Like LEGO blocks.

JavaScript

Programming language that makes websites interactive. The brain of the frontend.

JSON

JavaScript Object Notation - A way to organize data. Like a recipe card with labeled sections.

JWT

JSON Web Token - An encrypted token that proves you're logged in. Like a digital wristband.

PIN

Personal Identification Number - A secret code to prove who you are.

Request

Asking the server for something. Like ordering food at a restaurant.

Response

The server's answer to your request. Like receiving your food order.

SQL

Structured Query Language - Language for talking to databases.

Token

A piece of data that proves you're authenticated. Like a ticket or wristband.

Variable

A named storage location. Like a labeled box where you keep things.
